# JavaScript Todo Scheduler

A modern and responsive **Task Scheduling Web Application** built using **HTML, CSS, and Vanilla JavaScript**.

Designed with a clean UI, smooth animations, and structured code architecture, this project demonstrates strong frontend fundamentals without using any frameworks.

---

##  Overview

Pro Tasks is a lightweight task management app that allows users to:

- Add tasks with scheduled date & time
- Mark tasks as completed
- Delete individual tasks
- Clear all tasks
- Experience smooth UI animations and modern styling

The project focuses on dynamic DOM manipulation and interactive UI behavior using pure JavaScript.

---

##  Tech Stack

- **HTML5** – Semantic structure  
- **CSS3** – Modern UI (Flexbox, gradients, glass-style card, transitions)  
- **JavaScript (ES6)** – DOM manipulation & event handling  
- **Font Awesome** – Icons  

---

## Features

-  Add tasks with `datetime-local` support  
-  Calendar icon interaction  
-  Mark tasks as completed  
-  Delete individual tasks  
-  Clear all tasks  
-  Modern gradient background  
-  Smooth fade-in animations
